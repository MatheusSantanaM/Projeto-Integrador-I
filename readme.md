# Sem Auto Escola

## Descrição

Sem Auto Escola é um jogo educativo de simulação de trânsito desenvolvido para ensinar regras de circulação de forma interativa, prática e segura. O projeto permite que o jogador enfrente situações do cotidiano no trânsito, tome decisões e receba feedback sobre suas ações ao longo das fases.

## Conceito do jogo

O jogo foi idealizado como um simulador educacional de trânsito, com foco no aprendizado das regras por meio da experiência prática em fases. A proposta é tornar o estudo do trânsito mais atrativo e acessível, simulando sinalizações, obstáculos, escolhas do condutor e consequências de decisões corretas ou incorretas.

O jogador progride por diferentes fases, cada uma representando situações que podem ocorrer no cotidiano de motoristas. Durante a jogabilidade, o sistema avalia as decisões tomadas e apresenta feedback imediato, reforçando o aprendizado de maneira lúdica.

## Funcionalidades
As principais funcionalidades do projeto, são:
- Níveis de dificuldade em: Fácil, Médio e Difícil.
- Seleção de carros, permitindo escolher modelos e cores sem alterar a lógica do jogo.
- Menu de seleção de fases com visualização de fases disponíveis.
- Tela de resultados ao fim de cada fase, informando acertos e infrações cometidos.
- Menu de pausa para interromper a partida sem perder o progresso atual.
- Velocímetro nas telas de fase para indicar a velocidade atual do veículo.
- Trilha sonora e efeitos sonoros para ambientação e feedback de ações.

## Como executar

1. Baixe e instale o **GameMaker**.
2. Clone o repositório do projeto com `git clone https://github.com/MatheusSantanaM/Projeto-Integrador-I.git`.
2. Abra o projeto no **GameMaker**.
3. Localize e execute o arquivo **Simulador de Trânsito**.
4. Execute o jogo com **F5** ou no botão **Run** na parte superior da engine.
